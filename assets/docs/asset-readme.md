# Nestory Asset Pack v0.1

## Brand

Name: Nestory  
Tagline: Shared life, softly organized.  
Direction: cozy, romantic, playful, privacy-aware relationship planner.

## Included

- `svg/nestory-app-icon.svg`
- `svg/nestory-logo-horizontal.svg`
- `svg/nestory-wordmark.svg`
- `svg/nestory-splash.svg`
- decorative blob SVGs
- small feature/charm icons
- React Native theme tokens
- React Native primitive components

## UI shape language

Nestory should avoid plain rounded-box SaaS UI. Use:
- Pebble cards
- Nest panels
- Ribbon headers
- Charm buttons
- Calendar bubbles
- Gentle decorative blobs

## Implementation note

These assets are intentionally SVG-first so they can be edited easily and exported into Android densities later.
For React Native, use `react-native-svg` or convert final app icons through Android Studio / asset generator.
